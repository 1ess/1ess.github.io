本篇，我们介绍一种用于超快检索的数据结构 —— 哈希表(Hash Tables)或者称为字典(Dictionary)。

几乎所有的编程语言都有对应的实现，例如 Java 中的 HashMap、JavaScript 中的 Object、C# 中的 Dictionary。

Hash Collision

哈希函数会使用哈希算法根据输入计算出哈希表的 Value 实际存储的索引，可能发生不同输入计算出同一索引值的情况，这种我们就称之为哈希碰撞。

比较常用的有两种方式解决哈希碰撞:

  1. Chaining
  2. Open Addressing
